Das System ist eingeschaltet und bereit zur Verabreichung einer Dosis. Durch zweimaliges Drücken der Dosiertaste oben auf dem System innerhalb von 3 Sekunden kann eine Dosis verabreicht werden. In diesem Modus wird erst dann Fentanyl freigesetzt, wenn eine Dosis ausgelöst wird. Nach 24 Stunden oder 80 Dosisgaben schaltet sich das System ab. Wenn der Selbsttest des Systems fehlschlägt oder die Batterie schwach ist, ertönt ein SIGNAL und das System schaltet sich ab.

Freisetzung einer Dosis Fentanyl. Über einen Zeitraum von 10 Minuten werden 40 Mikrogramm des Medikaments an den Patienten verabreicht. Während dieser Zeit kann keine weitere Dosis ausgelöst werden. Die Leuchtanzeige blinkt grün und im Display wird angezeigt, dass gerade eine Dosis verabreicht wird. Im Display erscheinen abwechselnd eine rotierende LCD-Animation und die Anzahl der bereits verabreichten Dosen. Sollte das System einen schlechten Hautkontakt feststellen, beendet es die Verabreichung, wechselt in den Standby-Modus und kehrt dann in den „Bereit“-Modus zurück. Nach 24 Stunden oder 80 Dosisgaben wechselt das System in den „Ende der Anwendung“-Modus. Wenn das System eine Hautschädigung feststellt (z. B. eine offene Wunde), ertönt ein SIGNAL und es wechselt in den „Ende der Lebensdauer“-Modus.

Wenn eine Dosis ausgelöst wird, das System ABER nach 4 Minuten KEINEN angemessenen Hautkontakt beim Patienten feststellen kann, wird die Dosisfreisetzung beendet. Das System wechselt in den „Standby“-Modus und der Patient wird informiert. Dies wird angezeigt durch einen Piepton, während die Leuchtanzeige gleichzeitig rot blinkt. Das System wechselt für 15 Sekunden in den „Standby“-Modus. Die Anzahl der verabreichten Dosen erhöht sich nicht und das System kann erneut eine Dosis freisetzen, nachdem es nach 15 Sekunden wieder in den „Bereit“-Modus gewechselt ist.

Nachdem 80 Dosen verabreicht wurden oder 24 Stunden seit der Aktivierung vergangen sind, wechselt das System in den „Ende der Anwendung“-Modus. Das System setzt keine weiteren Dosen mehr frei, im LCD-Display blinkt die Gesamtzahl der verabreichten Dosen und die Leuchtanzeige blinkt nicht. Nach 12 Stunden im „Ende der Anwendung“-Modus wechselt das System automatisch in den „Ende der Lebensdauer“-Modus. Durch Drücken der Dosiertaste über einen Zeitraum von 6 Sekunden wird das System ausgeschaltet.

Ein ALARM-Zustand kann ausgelöst werden, wenn während der Verabreichung einer Dosis eine Hautschädigung festgestellt wird, die Batterie schwach ist oder wenn seit dem Erreichen des „Ende-der-Anwendung“-Modus 12 Stunden vergangen sind. Im LCD-Display blinkt die Anzahl der verabreichten Dosen, die Leuchtanzeige blinkt dauerhaft rot und es ertönt ein Dauerpiepton. Dies zeigt an, dass das System das Ende der Lebensdauer erreicht hat. Es kann keine Dosis mehr verabreicht werden. Durch Drücken der Dosiertaste über einen Zeitraum von 6 Sekunden wird das System ausgeschaltet.

IONSYS® darf nur in einem Krankenhaus angewendet werden. Die Behandlung sollte nur durch einen Arzt eingeleitet werden, der Erfahrung in der Durchführung einer Opioid-Therapie hat, und unter dessen Aufsicht erfolgen.

SYSTEMSPEZIFIKATIONEN
Wirkstoff Fentanylhydrochlorid
Absorptionsweg: Über die Haut (transdermal, Iontophorese)
Aktivierung: Der Patient drückt (innerhalb von 3 Sekunden) zweimal auf die Taste zur Bedarfsdosierung
Dosierung: Über einen vorprogrammierten Zeitraum von 10 Minuten werden 40 Mikrogramm des Wirkstoffs freigesetzt*
Maximale Dosisrate: 6 Dosen pro Stunde
Sperrzeit*:

10 Minuten (deckt sich mit dem Dosierungszeitraum)

Verfügbare Dosen pro System: 80
Anwendungsdauer: 24 Stunden oder 80 Dosen, je nachdem was zuerst erreicht ist

* (in dieser App wurde der 10-minütige Zeitraum zu Demonstrationszwecken auf 18 Sekunden gekürzt)

FUNKTIONEN
  • Taste zur Bedarfsdosierung
  • Leuchtdiode (LED) zeigt Folgendes an:
    • Wenn eine Dosis freigesetzt wird (grün)
    • Wenn ein Fehler aufgetreten ist (rot)
  • LCD-Display zeigt Folgendes an:
    • Anzahl der verabreichten Dosen
    • Wenn eine Dosis freigesetzt wird
    • Wenn ein Fehler aufgetreten ist
  • Akustische Signale ertönen bei Dosisaktivierung und bei verschiedenen Systemwarnungen
  • Druckempfindliche Klebefläche, die IONSYS® sicher auf der Haut des Patienten fixiert.
